# hello.rb
![](https://github.com/gowda/hello.rb/workflows/lint-and-tests/badge.svg)

'hello world' web application using [`rack`](https://github.com/rack/rack)

## Run
### Lint & tests
```bash
$ bin/rake cop
$ bin/rake spec
```
### Server
```bash
$ bin/rack server
```

Server runs at [http://localhost:9292](http://localhost:9292) by default
